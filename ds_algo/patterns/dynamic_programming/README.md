# Dynamic Programming Patterns

## Patterns

| # | Pattern | Key Idea | Classic Problems |
|---|---------|----------|-----------------|
| 01 | Fibonacci | Current state from previous 1-2 states | Climbing Stairs, House Robber, Decode Ways |
| 02 | 0/1 Knapsack | Include/exclude each item once | Subset Sum, Partition Equal Subset, Target Sum |
| 03 | Unbounded Knapsack | Items can be reused | Coin Change, Rod Cutting, Word Break |
| 04 | Longest Common Subsequence | Two-sequence alignment | LCS, Edit Distance, Min Deletions |
| 05 | Longest Increasing Subsequence | Single-sequence ordering | LIS, Russian Doll Envelopes, Max Chain Length |
| 06 | Kadane's | Max/min contiguous subarray | Max Subarray, Max Product Subarray, Circular Subarray |
| 07 | Matrix Chain Multiplication | Optimal split point in a range | MCM, Burst Balloons, Min Score Triangulation |
| 08 | Palindromic Subsequence | Palindrome structure on subsequences | Longest Palindromic Subseq, Min Insertions |
| 09 | DP on Trees | Post-order traversal + DP | Tree Diameter, Max Path Sum, House Robber III |
| 10 | DP on Grids | 2D traversal with state | Unique Paths, Min Path Sum, Dungeon Game |
| 11 | Bitmask DP | State = bitmask of visited/used items | TSP, Partition to K Equal Subsets, Shortest Superstring |
| 12 | Interval DP | Merge/split intervals optimally | Stone Game, Remove Boxes, Strange Printer |
| 13 | Digit DP | Count numbers with digit constraints | Numbers At Most N Given Digit Set, Count Special Integers |
| 14 | State Machine DP | Transitions between states | Best Time to Buy/Sell Stock (all variants), Student Attendance |

## How to use

Each directory contains:
- `pattern.md` — explanation, template, and complexity
- Solution files for classic problems
