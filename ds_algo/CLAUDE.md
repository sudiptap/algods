# DS & Algo Interview Prep - Project Settings

## Permissions
- Create and edit markdown files freely
- Create and edit code files (Python, Java, etc.) freely
- Run code for testing solutions
- NO destructive commands (rm, rm -rf, git reset --hard, etc.)
- NO force pushes or branch deletions

## Workflow
- Each pattern has its own markdown file in `patterns/`
- Leetcode problems are cataloged in the pattern file they belong to
- Solutions go in `solutions/` organized by pattern
- Use Python as the default language unless specified otherwise

## Conventions
- Pattern files: `patterns/<pattern-name>.md`
- Solution files: `solutions/<pattern-name>/<problem-number>-<problem-name>.py`
- Each solution includes: problem description, approach, time/space complexity, and code
